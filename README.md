# AES__Encryption

2020年新工科联盟-Xilinx暑期学校（Summer School）项目。

通过vivado2018.2实现简单的AES算法加密，并通过UART串口输出

这种加密主要运用在各种数据的传输之中，意为保护客户的以及信息的安全。其安全性相当可靠，在国家安全，机密数据传输等方面都有广泛的应用。
我们做的AES加密程序支持输入四位密钥的八位数据加密，通过UART串口输出加密后的128位数据。

共两个工程，其中AES_Encryption为满足实用条件的设计，包含全部功能，AES_Encryption_alt为为适应硬件以及观察现象对输入输出进行精简的项目。
